Combat YouTube Addiction
Main ⁜ Facebook ⁜ YouTube

Guide

First, set your limit on for how much time you can visit YouTube per day (daily limit < 2 hours). Next, set your limit on for how much time you can visit YouTube in one sitting (session limit < 1 hour). Last, set how much time you must wait in between sessions (minimum interval > 1 hour). Once either the daily limit or the session limit has run out, this extension will vex you whenever you watch YouTube videos.
